# Personal Finance Tracker

Personal finance tracker built in Excel — part of my data analysis learning journey.

## About
Spreadsheet built with real personal financial data tracking 12 months of expenses,
comparing actual spending against monthly targets.

## Features
- 12 months of real expense data
- Monthly spending vs. target comparison
- Automatic difference calculation
- Conditional formatting (🔴 over budget / 🟢 under budget)

## Tools
- Microsoft Excel Online
